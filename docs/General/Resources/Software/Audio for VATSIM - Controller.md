## What is Audio for VATSIM?


Congratulations! You are now a fully fledged VATSIM controller. Now we need to talk to the aircraft we will be controlling on the network. In VATSSA we use TrackAudio as our recommended audio client. A controller guide for TrackAudio can be found [here](https://eaip2.vatssa.com/General/Resources/Software/TrackAudio%20Guide%20-%20Controller/).

Our resources page, where you can find the TrackAudio download link can be found [here](https://eaip2.vatssa.com/General/Resources/Links/)
